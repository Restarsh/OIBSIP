# Personal Portfolio Website

## Overview
This is a static personal portfolio website showcasing skills, projects, and contact information. Built with HTML5 and CSS3 (no JavaScript required), it serves as a digital résumé.

## Files
- `index.html` – Main portfolio page containing Hero, About, Skills, Projects, and Contact sections.
- `style.css` – Stylesheet defining layout, colours, typography, and responsive behaviour.

## Features
- **Profile/Hero Section**: Name, role title, and avatar placeholder.
- **About Me**: Brief background and interests.
- **Skills**: Visual grid of technical skills (HTML5, CSS3, JavaScript, React, Node.js, Python, Git, REST APIs).
- **Projects**: At least two placeholder project cards with title, description, and GitHub‑link placeholders.
- **Contact**: Name, email (mailto:), LinkedIn and GitHub links.
- **Smooth Scroll**: Enabled via `scroll-behavior: smooth`.
- **Consistent Branding**: Unified colour palette and font family (Poppins from Google Fonts).
- **Fully Responsive**: Layout adapts using Flexbox/Grid and media queries (≤768 px).
- **No Overlap**: Intentional padding, margin, and `box-sizing: border-box`.

## How to View
1. Open the `personal portfolio` folder.
2. Double‑click `index.html` to open it in your default web browser.
3. Use the top navigation bar to jump between sections, or scroll smoothly.

## Customisation
- **Avatar**: Replace the placeholder image URL in `<img src="...">` with your own photo.
- **Text**: Edit the copy in `index.html` to reflect your actual bio, skills, projects, and contact details.
- **Colours**: Adjust the CSS variables (`--primary-color`, `--accent-color`, etc.) at the top of `style.css`.
- **Fonts**: Change the Google Fonts link or `font-family` in `style.css`.
