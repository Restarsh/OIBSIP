# AlexMan Landing Page

## Project Overview
This is a simple static landing page for **AlexMan**, an exceptional coffee brand focused on ethically sourced, sustainably hand-roasted coffee. The project demonstrates foundational HTML5 and CSS3 skills, including:

- Fixed/sticky navigation bar
- Hero section with headline, sub‑headline, and call‑to‑action button
- Multiple content sections (Features, About, Testimonials)
- Footer with contact information and social‑media placeholders
- Consistent colour palette using CSS variables
- Responsive layout using CSS Grid and Flexbox (mobile‑friendly down to 768px)
- Proper spacing, margins, and `box-sizing: border-box` to avoid element overlap
- Clear typographic hierarchy (different heading and body font sizes)

## Files
- `index.html` – Home page
- `about.html` – About AlexMan (story and values)
- `contact.html` – Contact form and contact details
- `style.css` – Shared stylesheet for all pages

## Responsive Design
The layout adapts to narrower screens:
- Navigation links wrap and center on screens ≤768 px.
- Hero text scales down for better readability.
- Grid columns re‑flow to a single column on small devices.

## Customisation
- **Colours**: Edit the CSS variables in `:root` at the top of `style.css`.
- **Text**: Update the copy in any HTML file.
- **Images**: Replace the Unsplash placeholder URLs in the inline hero sections (or add `<img>` tags) with your own assets.
- **Form Action**: The contact form currently uses `action="#"`. Replace with your form‑handling endpoint.
